LINE Messaging API

SN Checker Bot
Setup Guide

ทำตามขั้นตอนนี้ครบ ได้ bot พร้อมใช้งานใน ~30 นาที

📱 LINE Account → ⚙️ Messaging API → ☁️ Deploy Railway → 🔗 Set Webhook → ✅ ใช้งานได้


1

📁 เตรียม File โปรเจกต์

วาง 5 ไฟล์นี้ไว้ใน folder เดียวกัน:

app.py
Logic หลักของ bot
requirements.txt
Library ที่ต้องใช้
Procfile
คำสั่ง start server
438_Unit.xlsx
ไฟล์ข้อมูล SN ของคุณ
⚠️ ชื่อ column SN ในไฟล์ Excel คืออะไร? เปิดดูก่อน แล้วแก้ในไฟล์ .env.example บรรทัด SN_COLUMN=
2

💚 สร้าง LINE Official Account

  1. ไปที่ developers.line.biz → Login ด้วย LINE ส่วนตัว
  2. กด Create a new provider → ตั้งชื่อ (เช่น "SN Checker")
  3. กด Create a Messaging API channel
  4. กรอกข้อมูล → สร้างเสร็จ
  5. ไปที่แท็บ Messaging API → Copy Channel Secret
  6. กด Issue ใต้ Channel access token → Copy Access Token
⚠️ เก็บ token ทั้งสองไว้ก่อน จะใช้ใน Step 4
3

☁️ Deploy ขึ้น Railway (ฟรี)

  1. ไปที่ railway.app → Sign up ด้วย GitHub
  2. กด New Project → Deploy from GitHub repo
  3. อัป folder ทั้งหมดขึ้น GitHub ก่อน (ถ้ายังไม่มีให้สร้าง private repo)
  4. เลือก repo ที่สร้าง → Railway จะ build อัตโนมัติ
⚠️ อย่า push ไฟล์ .env จริงขึ้น GitHub! ใส่ token ใน Railway dashboard แทน (ดู Step 4)
4

🔑 ตั้งค่า Environment Variables ใน Railway

ใน Railway dashboard → Project → กด Variables → เพิ่มทีละบรรทัด:

LINE_CHANNEL_ACCESS_TOKEN = ใส่ token ที่ copy จาก Step 2
LINE_CHANNEL_SECRET       = ใส่ secret ที่ copy จาก Step 2
SN_COLUMN                 = SN   # ← เปลี่ยนให้ตรง header ของไฟล์
EXCEL_FILE                = 438_Unit.xlsx
✅ Railway จะ restart bot ให้อัตโนมัติหลังบันทึก
5

🔗 ตั้งค่า Webhook ใน LINE

หลัง deploy สำเร็จ Railway จะให้ URL เช่น:

https://your-app-name.up.railway.app

กลับไป LINE Developers → แท็บ Messaging API:

  1. Webhook URL → ใส่ https://your-app-name.up.railway.app/webhook
  2. กด Verify → ต้องขึ้น "Success"
  3. เปิด Use webhook → ON
  4. ปิด Auto-reply messages → OFF
6

✅ ทดสอบ Bot

เพิ่ม bot เป็นเพื่อนใน LINE (QR code อยู่ใน LINE Developers) แล้วพิม SN ส่งไป:

# ถ้า SN อยู่ในไฟล์:
⚠️ SN: ABC123456
Status: need controller replacement

# ถ้า SN ไม่อยู่ในไฟล์:
✅ SN: XYZ999
ไม่พบในรายการ — ปกติ
★

🔄 อัปเดตข้อมูล SN ทีหลัง

ถ้าไฟล์ Excel มีการเพิ่ม/ลบ SN ทีหลัง มี 2 วิธี:

  1. Push ไฟล์ใหม่ขึ้น GitHub → Railway rebuild อัตโนมัติ (แนะนำ)
  2. เรียก URL /reload เพื่อ reload ข้อมูลโดยไม่ restart

🎉 เสร็จแล้ว!

พิม SN ใน LINE → Bot เช็คจาก Excel → ตอบกลับทันที
ถ้าติดปัญหาขั้นตอนไหน บอกได้เลยครับ
